# 📌 Project Overview

This project presents a **Sales Performance Dashboard** developed using **Power BI**, **MySQL**, and **Python**.

The dashboard transforms raw sales data into meaningful business insights through interactive reports, KPIs, DAX measures, SQL analysis, and professional visualizations.

---

# 🎯 Project Objectives

- Analyze total sales performance
- Compare regional sales
- Identify top-selling products
- Analyze customer behavior
- Monitor monthly sales trends
- Track payment methods
- Measure sales achievement
- Support business decision-making using data
