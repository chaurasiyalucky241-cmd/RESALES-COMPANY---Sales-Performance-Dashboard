# 📌 Project Overview

The **RESALES COMPANY – Sales Performance Dashboard** is a Business Intelligence project that transforms raw sales data into meaningful insights using **Power BI**, **MySQL**, and **Python**.

The dashboard helps businesses monitor sales performance, customer behavior, regional sales, payment methods, shipping costs, and product performance through interactive visualizations.

---

# 🎯 Project Objectives

- Analyze overall sales performance
- Monitor monthly sales trends
- Compare regional sales
- Identify top-selling products
- Analyze customer behavior
- Monitor shipping costs
- Analyze payment methods
- Build an interactive dashboard
- Support business decision-making
